# Система Дизайну, Натхненна Pinterest

> Category: Media & Consumer
> Візуальне відкриття. Червоний акцент, сітка masonry, зображення на першому плані.

## 1. Візуальна Тема та Атмосфера

Вебсайт Pinterest — це тепле, надихаюче полотно, яке перетворює візуальне відкриття на стиль мисленнєвого журналу. Дизайн побудований на м'якому, злегка теплому білому тлі, де Pinterest Red (`#e60023`) є єдиним сміливим акцентом бренду. На відміну від холодних відтінків синього більшості технологічних платформ, нейтральна шкала Pinterest має виразно теплий підтон — сірі відтінки тяжіють до оливково-піщаних тонів (`#91918c`, `#62625b`, `#e5e5e0`) замість холодної сталі, утворюючи затишну, ремісничу атмосферу, яка запрошує до перегляду.

Типографіка використовує Pin Sans — власний фірмовий шрифт із широким стеком запасних варіантів, включаючи японські шрифти, що відображає глобальний охоплення Pinterest. На відображальному масштабі (70px, жирність 600) Pin Sans створює великі, привабливі заголовки. На менших розмірах система компактна: кнопки 12px, підписи 12–14px. Система іменування CSS-змінних (`--comp-*`, `--sema-*`, `--base-*`) розкриває складну трирівневу архітектуру токенів дизайну: токени рівня компонента, семантичного рівня та базового рівня.

Pinterest вирізняється щедрою системою border-radius (12px–40px, плюс 50% для кіл) і теплими фонами кнопок. Другорядна кнопка (`#e5e5e0`) має виразно теплий, піщаний тон, а не холодний сірий. Основна червона кнопка використовує радіус 16px — округла, але не пігулкоподібна. У поєднанні з теплими фонами значків (`hsla(60,20%,98%,.5)` — ледь помітна жовто-тепла підсвітка) та макетами, де домінує фотографія, результатом є дизайн, що відчувається як рукотворний і особистісний, а не корпоративний і стерильний.

**Ключові Характеристики:**
- Тепле біле полотно з оливково/піщаними нейтральними тонами — затишне, не клінічне
- Pinterest Red (`#e60023`) як єдиний сміливий акцент — ніколи не ненав'язливий, завжди впевнений
- Фірмовий шрифт Pin Sans із глобальним стеком запасних варіантів (включаючи CJK)
- Трирівнева архітектура токенів: `--comp-*` / `--sema-*` / `--base-*`
- Теплі другорядні поверхні: піщаний сірий (`#e5e5e0`), тепла нашивка (`hsla(60,20%,98%,.5)`)
- Щедрий border-radius: стандартний 16px, до 40px для великих контейнерів
- Контент, де зображення на першому місці — пін/зображення є основним візуальним елементом
- Темно-майже-фіолетовий текст (`#211922`) — теплий, з відтінком сливи

## 3. Правила Типографіки

### Сімейство Шрифтів
- **Основний**: `Pin Sans`, запасні: `-apple-system, system-ui, Segoe UI, Roboto, Oxygen-Sans, Apple Color Emoji, Segoe UI Emoji, Segoe UI Symbol, Ubuntu, Cantarell, Fira Sans, Droid Sans, Helvetica Neue, Helvetica, ヒラギノ角ゴ Pro W3, メイリオ, Meiryo, ＭＳ Ｐゴシック, Arial`

### Ієрархія

| Роль | Шрифт | Розмір | Жирність | Висота рядка | Міжлітерний інтервал | Примітки |
|------|-------|--------|---------|--------------|---------------------|---------|
| Відображальний Герой | Pin Sans | 70px (4.38rem) | 600 | нормальний | нормальний | Максимальний ефект |
| Заголовок секції | Pin Sans | 28px (1.75rem) | 700 | нормальний | -1.2px | Від'ємне трекінгування |
| Основний текст | Pin Sans | 16px (1.00rem) | 400 | 1.40 | нормальний | Стандартне читання |
| Жирний підпис | Pin Sans | 14px (0.88rem) | 700 | нормальний | нормальний | Виразні метадані |
| Підпис | Pin Sans | 12px (0.75rem) | 400–500 | 1.50 | нормальний | Малий текст, теги |
| Кнопка | Pin Sans | 12px (0.75rem) | 400 | нормальний | нормальний | Мітки кнопок |

### Принципи
- **Компактна типографічна шкала**: Діапазон — від 12px до 70px із драматичним стрибком; більшість функціонального тексту — 12–16px, утворюючи щільну ієрархію інформації на кшталт додатка.
- **Теплий розподіл жирності**: 600–700 для заголовків, 400–500 для основного тексту. Без ультралегких жирностей — типографіка завжди відчувається вагомою.
- **Від'ємне трекінгування в заголовках**: -1.2px у заголовках 28px створює затишні, інтимні назви секцій.
- **Одне сімейство шрифтів**: Pin Sans обслуговує все — не виявлено другорядного відображального або моноширинного шрифту.

## 4. Стилі Компонентів

### Кнопки

**Основний Червоний**
- Фон: `#e60023` (Pinterest Red)
- Текст: `#000000` (чорний — незвичний вибір для контрасту на червоному)
- Відступ: 6px 14px
- Радіус: 16px (щедро округлений, не пігулкоподібний)
- Межа: `2px solid rgba(255, 255, 255, 0)` (прозора)
- Фокус: семантична межа + обведення через CSS-змінні

**Другорядний Піщаний**
- Фон: `#e5e5e0` (теплий піщаний сірий)
- Текст: `#000000`
- Відступ: 6px 14px
- Радіус: 16px
- Фокус: та сама семантична система меж

**Кругла Дія**
- Фон: `#e0e0d9` (тепле світло)
- Текст: `#211922` (сливовий чорний)
- Радіус: 50% (коло)
- Використання: дії з пінами, елементи навігації

**Ghost / Прозорий**
- Фон: прозорий
- Текст: `#000000`
- Без межі
- Використання: третинні дії

### Картки та Контейнери
- Картки пінів із пріоритетом фотографії та щедрим радіусом (12px–20px)
- Немає традиційної box-shadow на більшості карток
- Білі або теплі туманні фони
- Товста біла межа 8px на деяких контейнерах зображень

### Поля Введення
- Введення email: білий фон, межа `1px solid #91918c`, радіус 16px, відступ 11px 15px
- Фокус: семантична система меж і обведення через CSS-змінні

### Навігація
- Чистий заголовок на білому або теплому фоні
- Логотип Pinterest + рядок пошуку по центру
- Pin Sans 16px для навігаційних посилань
- Акценти Pinterest Red для активних станів

### Обробка Зображень
- Сітка masonry у стилі пінів (фірмовий макет Pinterest)
- Скруглені кути: 12px–20px на зображеннях
- Фотографія як основний контент — кожен пін є зображенням
- Товсті білі межі (8px) на виділених контейнерах зображень

## 5. Принципи Макету

### Система Відступів
- Базова одиниця: 8px
- Шкала: 4px, 6px, 7px, 8px, 10px, 11px, 12px, 16px, 18px, 20px, 22px, 24px, 32px, 80px, 100px
- Великі стрибки: 32px → 80px → 100px для відступів між секціями

### Сітка та Контейнер
- Сітка masonry для контенту пінів (фірмовий макет)
- Центровані секції контенту із щедрою максимальною шириною
- Темний футер на всю ширину
- Рядок пошуку як основний елемент навігації

### Філософія Пустого Простору
- **Щільність натхнення**: Сітка masonry щільно пакує піни — щільність контенту є ціннісною пропозицією. Пустий простір існує між секціями, але не всередині сітки.
- **Дихання зверху, щільність знизу**: Секції героя/функцій отримують щедрий відступ; сітка пінів компактна й занурююча.

### Шкала Border Radius
- Стандартний (12px): Малі картки, посилання
- Кнопка (16px): Кнопки, поля введення, середні картки
- Комфортний (20px): Картки функцій
- Великий (28px): Великі контейнери
- Секція (32px): Елементи вкладок, великі панелі
- Герой (40px): Контейнери героїв, великі блоки функцій
- Коло (50%): Кнопки дії, індикатори вкладок

## 6. Глибина та Висота

| Рівень | Обробка | Використання |
|--------|---------|-------------|
| Плаский (Рівень 0) | Без тіні | За замовчуванням — піни покладаються на контент, а не на тінь |
| Ледь помітний (Рівень 1) | Мінімальна тінь (із токенів) | Піднятими накладки, спадні меню |
| Фокус (Доступність) | Кільце `--sema-color-border-focus-outer-default` | Стани фокусування |

**Філософія Тіней**: Pinterest використовує мінімальні тіні. Сітка masonry покладається на контент (фотографію) для створення візуального інтересу, а не на ефекти підняття. Глибина походить від теплоти кольорів поверхонь і щедрого скруглення контейнерів.

## 7. Що Робити і Чого Уникати

### Робити
- Використовуй теплі нейтральні тони (`#e5e5e0`, `#e0e0d9`, `#91918c`) — теплий оливково/піщаний тон є ідентичністю
- Застосовуй Pinterest Red (`#e60023`) лише для основних CTA — він сміливий і єдиний
- Використовуй Pin Sans виключно — один шрифт для всього
- Застосовуй щедрий border-radius: 16px для кнопок/полів введення, 20px+ для карток
- Зберігай сітку masonry щільною — щільність контенту є цінністю
- Використовуй теплі фони нашивок (`hsla(60,20%,98%,.5)`) для ледь помітних теплих промивань
- Використовуй `#211922` (сливовий чорний) для основного тексту — він тепліший за чистий чорний

### Уникати
- Не використовуй холодні сірі нейтральні тони — завжди теплий/оливковий відтінок
- Не використовуй чистий чорний (`#000000`) як основний текст — використовуй сливовий чорний (`#211922`)
- Не використовуй кнопки пігулкоподібної форми — радіус 16px округлений, але не пігулкоподібний
- Не додавай важких тіней — Pinterest плаский за своєю суттю, глибина від контенту
- Не використовуй малий border-radius (<12px) на картках — щедре скруглення є основним
- Не вводь додаткові кольори бренду — червоний + теплі нейтральні є повною палітрою
- Не використовуй тонкі жирності шрифту — Pin Sans мінімум 400

### Стратегія Згортання
- Сітка пінів: 5+ стовпців → 3 → 2 → 1

- Секції функцій: поруч → укладені
- Герой: 70px → масштабується пропорційно вниз
- Футер: темний багатоколонковий → укладений
