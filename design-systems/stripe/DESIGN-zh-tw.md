# 以 Stripe 為靈感的設計系統

> Category: 金融科技與加密貨幣
> 支付基礎設施。標誌性紫色漸層，字重 300 的優雅質感。

## 1. 視覺主題與氛圍

Stripe 的網站是金融科技設計的黃金標準——一套既能呈現技術感又兼具奢華感、既精準又不失溫度的系統。頁面以乾淨的白色畫布（`#ffffff`）為底，搭配深海軍藍標題（`#061b31`）與標誌性的紫色（`#533afd`），後者同時扮演品牌錨點與互動強調色的角色。這並非企業軟體那種冷冽、臨床的紫色，而是一種飽和度高的藍紫，傳遞出自信與高端的氣質。整體印象猶如一家金融機構經由世界頂尖字型鑄造廠重新詮釋。

自訂可變字型 `sohne-var` 是 Stripe 視覺識別的決定性元素。所有文字元素都啟用了 OpenType `"ss01"` 風格集，修改字型輪廓，呈現出鮮明的幾何現代感。在展示尺寸（48px–56px）下，sohne-var 採用字重 300——這對標題而言是極為纖細的字重，創造出一種飄渺而悄聲的權威感。這與「粗體英雄標題」的慣例截然相反；Stripe 的標題感覺不需要大聲呼喊。負值字距（56px 時為 -1.4px，48px 時為 -0.96px）將文字壓縮成緊密的工程感方塊。在較小尺寸下，系統同樣採用字重 300，並按比例縮減字距追蹤，且透過 `"tnum"` 啟用等寬數字，用於金融數據的呈現。

真正令 Stripe 與眾不同的是其陰影系統。有別於大多數網站的扁平或單層做法，Stripe 使用多層、帶藍色調的陰影：標誌性的 `rgba(50,50,93,0.25)` 搭配 `rgba(0,0,0,0.1)`，創造出帶有涼意、近乎大氣層深度的陰影——彷彿元素漂浮在暮色天空中。主陰影顏色（50,50,93）的藍灰色調直接呼應海軍藍-紫色的品牌調色盤，使連層次感都充滿品牌氣息。

**主要特徵：**
- 所有文字套用帶有 OpenType `"ss01"` 的 sohne-var——定義品牌字型輪廓的自訂風格集
- 字重 300 作為標誌性標題字重——輕盈、自信、反傳統
- 展示尺寸下的負值字距（56px 時為 -1.4px，向下按比例漸進放寬）
- 使用 `rgba(50,50,93,0.25)` 的藍色調多層陰影——帶有品牌色彩的層次感
- 深海軍藍（`#061b31`）標題取代黑色——溫潤、高端、金融質感
- 保守的圓角（4px–8px）——無膠囊形，無銳角
- 寶石紅（`#ea2261`）與洋紅（`#f96bee`）強調色，用於漸層與裝飾元素
- `SourceCodePro` 作為等寬字型搭檔，用於程式碼與技術標籤

## 2. 色彩調色盤與角色

### 主要色彩
- **Stripe 紫**（`#533afd`）：主要品牌色、CTA 背景、連結文字、互動強調。飽和的藍紫色，錨定整個系統。
- **深海軍藍**（`#061b31`）：`--hds-color-heading-solid`。主要標題色。非黑非灰——極深的藍色，為文字增添溫潤與深度。
- **純白**（`#ffffff`）：頁面背景、卡片表面、深色背景上的按鈕文字。

### 品牌色與深色
- **品牌深色**（`#1c1e54`）：`--hds-color-util-brand-900`。深靛藍，用於深色區塊、頁尾背景及品牌沉浸時刻。
- **深海軍藍**（`#0d253d`）：`--hds-color-core-neutral-975`。最深的中性色——近乎黑色但帶藍色調，呈現最大深度而不顯生硬。

### 強調色
- **寶石紅**（`#ea2261`）：`--hds-color-accentColorMode-ruby-icon-solid`。暖紅粉，用於圖示、警告與強調元素。
- **洋紅**（`#f96bee`）：`--hds-color-accentColorMode-magenta-icon-gradientMiddle`。鮮豔的粉紫，用於漸層與裝飾亮點。
- **淡洋紅**（`#ffd7ef`）：`--hds-color-util-accent-magenta-100`。洋紅主題卡片與徽章的帶色表面。

### 互動色
- **主紫**（`#533afd`）：主要連結色、啟用狀態、選取元素。
- **紫色懸停**（`#4434d4`）：主要元素懸停時的深紫色。

- **淡紫**（`#b9b9f9`）：`--hds-color-action-bg-subduedHover`。柔和薰衣草色，用於低調的懸停背景。

- **檸檬黃**（`#9b6829`）：`--hds-color-core-lemon-500`。警告與高亮強調色。

### 表面色與邊框
- **預設邊框**（`#e5edf5`）：卡片、分隔線與容器的標準邊框色。

### 原則
- **輕量字重作為標誌**：展示尺寸下的字重 300 是 Stripe 最具辨識度的字型選擇。當其他人用 600–700 吸引注意時，Stripe 以輕盈作為奢華的表達——文字如此自信，不需要字重來彰顯權威。
- **ss01 無所不在**：`"ss01"` 風格集是不可妥協的。它修改特定字型（可能是替代的 `a`、`g`、`l` 形式），為所有 sohne-var 文字創造更幾何、更當代的質感。
- **兩種 OpenType 模式**：展示/正文文字使用 `"ss01"`，金融數據的等寬數字使用 `"tnum"`。兩者永不重疊——段落中的數字使用 ss01，數據表格中的數字使用 tnum。
- **漸進式字距**：字距隨尺寸成比例收緊：56px 時 -1.4px、48px 時 -0.96px、32px 時 -0.64px、26px 時 -0.26px、16px 及以下 normal。
- **雙字重的簡潔**：主要使用 300（正文與標題）和 400（UI/按鈕）。主要字型不使用粗體（700）——SourceCodePro 以 500/700 提供程式碼對比。

## 5. 版面原則

### 間距系統
- 基礎單位：8px
- 刻度：1px、2px、4px、6px、8px、10px、11px、12px、14px、16px、18px、20px
- 特點：小端刻度緊密（4–12 每 2px 一格），反映 Stripe 針對金融數據的精準 UI 取向

### 網格與容器
- 最大內容寬度：約 1080px
- 英雄區：居中單欄，充裕內距，輕盈標題
- 功能區塊：2–3 欄網格的功能卡片
- 品牌沉浸式全寬深色區塊，背景色 `#1c1e54`
- 程式碼/儀表板預覽以帶藍色調陰影的容器卡片呈現

### 留白哲學
- **精準間距**：有別於極簡系統的大量空白，Stripe 使用有節制、有目的的留白。每個間隙都是刻意的排版選擇。
- **密集數據，寬裕框架**：金融數據呈現（表格、圖表）緊湊排列，周圍的 UI 框架則大方留白。這營造出一種受控的密度感——如同精緻框架中的一份整潔試算表。
- **區塊節奏**：白色區塊與深色品牌區塊（`#1c1e54`）交替出現，形成戲劇性的明暗韻律，避免單調而不引入隨意的色彩。

### 圓角刻度
- 微型（1px）：精細元素，細膩圓角
- 標準（4px）：按鈕、輸入框、徽章、卡片——主力
- 舒適（5px）：標準卡片容器
- 寬鬆（6px）：導覽、較大互動元素
- 大型（8px）：精選卡片、英雄元素
- 複合：`0px 0px 6px 6px` 用於底部圓角容器（分頁面板、下拉選單頁尾）

## 6. 深度與層次

| 層級 | 處理方式 | 用途 |
|-------|-----------|-----|
| 扁平（層級 0） | 無陰影 | 頁面背景、行內文字 |
| 環境（層級 1） | `rgba(23,23,23,0.06) 0px 3px 6px` | 細微卡片浮起、懸停提示 |
| 標準（層級 2） | `rgba(23,23,23,0.08) 0px 15px 35px` | 標準卡片、內容面板 |
| 高架（層級 3） | `rgba(50,50,93,0.25) 0px 30px 45px -30px, rgba(0,0,0,0.1) 0px 18px 36px -18px` | 精選卡片、下拉選單、彈出框 |
| 深度（層級 4） | `rgba(3,3,39,0.25) 0px 14px 21px -14px, rgba(0,0,0,0.1) 0px 8px 17px -8px` | 模態框、浮動面板 |
| 光環（無障礙） | `2px solid #533afd` 外框 | 鍵盤聚焦光環 |

**陰影哲學**：Stripe 的陰影系統建立在色彩深度的原則上。當大多數設計系統使用中性灰或黑色陰影時，Stripe 的主要陰影色（`rgba(50,50,93,0.25)`）是一種深藍灰，呼應品牌的海軍藍調色盤。這使陰影不只是增添深度——更增添品牌氛圍。多層做法將這種藍色調陰影與偏移不同的純黑次要層（`rgba(0,0,0,0.1)`）搭配，創造出視差般的深度：品牌色陰影距元素較遠，中性陰影距元素較近。負擴散值（-30px、-18px）確保陰影不會水平延伸超出元素範圍，使層次感保持垂直且受控。

### 裝飾深度
- 深色品牌區塊（`#1c1e54`）透過背景色對比創造沉浸式深度
- 寶石紅至洋紅過渡的漸層疊加，用於英雄裝飾
- 陰影色 `rgba(0,55,112,0.08)`（`--hds-color-shadow-sm-top`），用於黏性元素的頂邊陰影

## 7. 應做與不應做

### 應做
- 所有文字元素都使用帶有 `"ss01"` 的 sohne-var——風格集本身就是品牌
- 所有標題與正文文字使用字重 300——輕盈是標誌
- 所有高架元素套用藍色調陰影（`rgba(50,50,93,0.25)`）
- 標題使用 `#061b31`（深海軍藍）而非 `#000000`——溫潤感至關重要
- 將圓角保持在 4px–8px 之間——保守的圓角是刻意為之
- 所有等寬/金融數字顯示使用 `"tnum"`
- 疊加陰影：遠處藍色調 + 近處中性，創造深度視差
- 使用 `#533afd` 紫色作為主要互動/CTA 色

### 不應做
- 不要對 sohne-var 標題使用字重 600–700——字重 300 是品牌聲音
- 不要在卡片或按鈕上使用大圓角（12px+、膠囊形）——Stripe 風格保守
- 不要使用中性灰陰影——始終以藍色調（`rgba(50,50,93,...)`）染色
- 不要在任何 sohne-var 文字上省略 `"ss01"`——替代字型定義了品牌個性
- 不要對標題使用純黑（`#000000`）——始終使用 `#061b31` 深海軍藍
- 不要對互動元素使用暖色強調（橙色、黃色）——紫色是主色
- 不要在展示尺寸上套用正值字距——Stripe 字距緊湊
- 不要將洋紅/寶石紅強調色用於按鈕或連結——它們僅供裝飾/漸層使用

### 迭代指南
1. 始終在 sohne-var 文字上啟用 `font-feature-settings: "ss01"`——這是品牌的字型 DNA
