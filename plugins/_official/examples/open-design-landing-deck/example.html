Who hears this story · Nº 02

Three audiences, one honest thesis.

Partners evaluate the roadmap and the license, not the slide deck. Press wants one sentence they can quote. Community wants proof the story matches the commit log — so we write one narrative and let each audience pull what it needs.

  • Partners — read API stability, Apache-2.0 terms, and plugin roadmap before anything else.
  • Press — need a single clear thesis: local, open, agent-native design, stated plainly.
  • Community — 300+ contributors who fact-check every claim against the repo.

What we're actually offering · Nº 03

The offer is the belief, not a feature list.

We don't sell seats or gate the workflow. We open-source the plugins, the design systems, and the agent wiring itself — so the belief and the product are the same claim, provable in one clone.

  • Apache-2.0, permanently — no dual-license bait-and-switch later.
  • Local-first — your files and your own coding agent, not our cloud.
  • BYOK everywhere — Anthropic, OpenAI, Azure, Google, plus 14 media providers.
  • 217+ plugins, 129 design systems — the belief compounding in public.

Distribution channels · Nº 05

Five channels, the same thesis every time.

Each channel restates the identical belief in its native format, so a partner reading the README and a journalist reading the press note hear the same founder voice.

  • 01 · GitHub README + release notes — the thesis, restated at every ship.
