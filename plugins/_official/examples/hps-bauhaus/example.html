Why fundamentals first

Four habits, not opinions

Every senior designer at this studio started with the same four habits. Skip them and every later review turns into a fight about taste instead of a check against the rules:

  • Grid — every layout maps to an 8px baseline unit
  • Type — three sizes, one weight ramp, nothing improvised
  • Color — a 60/30/10 split, checked against WCAG AA
  • Critique — describe what you see before you say what you'd change

Module two · The Rules

Four numbers to know cold

8px
baseline grid unit — every margin, padding, and component height is a multiple
1.333
the type-scale ratio (Perfect Fourth) driving all six heading sizes
4.5:1
minimum contrast ratio for body text against its background, per WCAG AA
3
max accent colors on any one screen, on top of ink and paper
KPI grid. Numbers set in Archivo Black; each card takes one primary top rule, the fourth stays ink-only for rhythm.

Module three · Studio Workflow

The Friday critique loop

Look

Describe the layout in plain language before naming a fix — grid, type, color, hierarchy.

Measure

Pull the real numbers: baseline unit, type sizes, contrast ratios, color count.

Rebuild

Redo the weak component from scratch using only the four rules — no eyeballing.

Critique

Present the before/after to the pod; defend every measurement, not just the vibe.
