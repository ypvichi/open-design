The audit

When we opened the last 18 issues side by side, no two features shared a measure — column widths ranged from 38 to 64 characters, breaking the rhythm the culture desk had spent years building.

The masthead had been redrawn six times by six different freelancers, each shipping a slightly different kerning table with no single source file.

The fix isn't a font swap. It's a tri-tone system — dusty pink, mustard cream, deep burgundy — locked to one grid, so every desk ships the same magazine.

Eight rules, locked to the grid.

§ 03 — Rescue Principles
/ 01

Baseline grid

12 columns, 84px module. Every photo, pull-quote, and caption snaps to it — nothing floats.

/ 02

Three inks only

Dusty pink, mustard cream, deep burgundy. No fourth color enters a spread, ever.

/ 03

Two display faces

Bricolage Grotesque for headlines, Instrument Serif italic for kickers and pull-quotes.

/ 04

Masthead is fixed

One drawn wordmark, one kerning table. Freelancers inherit it — they don't redraw it.

/ 05

Caption discipline

Every caption in JetBrains Mono, uppercase, 0.15em tracking. Captions read as data, not decor.

/ 06

Measure ceiling

52 characters per line, hard ceiling. Wide features split two columns before they cross it.

/ 07

White space budget

Every spread reserves 18% dead space, minimum. Density is a choice, not a leftover.

/ 08

Section labels

§ 0X, top-right, mono, every spread. Readers always know where they are in the issue.

§ 04 — The Page System
One grid, four page types.

How the module works

Every spread is built from one 84px baseline module — the fixed unit for gutters, photo crops, and caption blocks. Four page types resolve to the same module, so pages 12 through 164 read as one object, not eighteen freelance interpretations.
